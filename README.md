### Setting up locally

- Clone the project and change directory.

  ```shell
  git clone https://github.com/arifszn/gitprofile.git
  cd gitprofile
  ```

- Install dependencies.

  ```shell
  npm install
  ```

- Run dev server.

  ```shell
  npm run dev
  ```

- Finally, visit `http://localhost:5173/gitprofile/` from your browser.
